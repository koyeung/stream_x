Extra Elixir Stream utilities

* e.g. `StreamX.transform` support switching reducer on-the-fly.

## Installation

* Add stream_x to your list of dependencies in `mix.exs`:

        def deps do
          [{:stream_x, "~> 0.0.1"}]
        end
